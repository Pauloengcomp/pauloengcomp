## Projects

### [price-data-pipeline](https://github.com/Pauloengcomp/price-data-pipeline)
Automated data pipeline for ingestion, processing and distribution of healthcare
pricing tables (Planserv, CMED/ANVISA and TUSS/ANS).

- RAW → SILVER → GOLD layered architecture
- Incremental update detection via state control
- CMED × TUSS enrichment by REGISTRO field join
- Automated delivery of processed files via Slack
- Orchestrated with Airflow, containerized with Docker + PostgreSQL
